Transition Properties:

  • transition-duration
  • transition-property
  • transition-timing-function
  • transition-delay

transition-duration

Specifies duration

  • transition-duration: 1s;
  • transition-duration: 0.5s;
  • transition-duration: 2s, 4s;

transition-property

Specifies which property should exactly change

  • transition-duration: 1s;
  • transition-duration: 0.5s;
  • transition-duration: 2s, 4s;

The order of transition-duration fits the order of transition-properties.

.animated {
transition-property: background-color, border-radius, transform;
transition-duration: 2s, 6s, 1s;
}
